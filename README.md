# Система учета товаров (CSV + GUI)

## Описание

Простая система для учета товаров, использующая CSV-файлы в качестве базы данных. Проект предоставляет как консольный интерфейс для выполнения основных операций (инициализация БД, резервное копирование, восстановление, анализ производительности), так и графический интерфейс пользователя (GUI) на базе `CustomTkinter` для интерактивного управления данными с поддержкой ролевого доступа.

## Возможности

*   **Управление данными:** Добавление, обновление и просмотр информации о товарах, категориях, поставщиках и поставках.
*   **CSV-хранилище:** Все данные хранятся в простых и легко читаемых CSV-файлах.
*   **Консольный интерфейс:**
    *   Инициализация базы данных с тестовыми данными.
    *   Создание резервных копий базы данных.
    *   Восстановление базы данных из резервной копии.
    *   Анализ производительности операций и генерация отчетов.
    *   Просмотр данных о товарах, категориях и поставщиках.
*   **Графический интерфейс (GUI):**
    *   Интуитивно понятный интерфейс на `CustomTkinter`.
    *   Система аутентификации с ролевым доступом (администратор, менеджер, просмотрщик).
    *   Возможность добавления и редактирования записей через GUI.
*   **Отчетность:** Генерация отчетов о производительности.

## Установка

Для запуска проекта необходимо установить Python 3.x и следующие зависимости:

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/MrBombastick08/Accounting-of-goods.git
    cd Accounting-of-goods
    ```
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Использование

### Консольный режим

Для запуска консольного меню выполните:

```bash
python main.py
```

Вам будет предложено выбрать одну из следующих операций:

*   `1` — Инициализация БД (создать CSV с тестовыми данными)
*   `2` — Резервное копирование
*   `3` — Восстановление из копии
*   `4` — Анализ производительности (отчёт)
*   `5` — Показать данные (товары с категорией и поставщиком)
*   `0` — Выход

### Графический интерфейс

Для запуска GUI выполните:

```bash
python app_gui.py
```

При первом запуске вам будет предложено войти в систему. Доступны следующие роли:

*   **admin**
*   **manager**
*   **view**

Пароли по умолчанию для этих ролей могут быть найдены в исходном коде или созданы при инициализации БД.
